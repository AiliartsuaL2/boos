# boos
Boos 프로젝트

# 개요
- 사업자들을 위한 정보공유 커뮤니티
  - 공공 데이터 포탈의 사업자 진위 확인 API를 이용하여 회원 가입을 합니다.
  - 일반적인 커뮤니티의 성격을 띄고 있습니다.

# 기술스택
- Server
  - Framework : SpringBoot 2.7.9
  - Language : Java 11
  - Security : Spring Security , JsonWebToken
  - JDBC : JPA , QueryDsl
- DataBase
  - AWS RDS
  - MySQL 8.0.28
- CI / CD
  - Jenkins
